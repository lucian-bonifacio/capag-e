**Badge** — small pill for status, counts, or categories (e.g. "Incluída", "Excluída", account counts).

```jsx
<Badge variant="success">Incluída</Badge>
<Badge variant="neutral">12 contas</Badge>
<Badge variant="warning">Revisar</Badge>
```

Variants: `neutral`, `primary`, `success`, `warning`, `danger`. Soft tinted background with matching text — never solid/saturated. Keep labels to one or two words.
