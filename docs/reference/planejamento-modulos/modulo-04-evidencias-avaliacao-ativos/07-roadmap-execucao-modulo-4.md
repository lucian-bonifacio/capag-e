# 07 - Roadmap de execucao do Modulo 04

## Sequencia proposta

1. Revisar `SPEC-07` e `SPEC-08`.
2. Definir entidade `AdjustmentEvidence`.
3. Definir materialidade.
4. Definir status de evidencia.
5. Definir entidade `AssetValuationAssessment`.
6. Definir classificacoes de realizabilidade.
7. Definir bloqueios de `PLRA`.
8. Definir bloqueios/ressalvas de laudo.
9. Planejar API de evidencias.
10. Planejar API de avaliacao de ativos.
11. Planejar telas de evidencias e ativos.
12. Planejar abas de exportacao.
13. Planejar testes por camada.
14. Revisar compatibilidade com Modulos 01, 03 e 07.

## Criterios de aceite

- `SPEC-07` e `SPEC-08` estao cobertas.
- Ajustes materiais possuem justificativa e evidencia.
- Ativos materiais possuem tratamento especifico.
- `PLRA` nao absorve ajuste material sem base, bloqueio ou ressalva.
