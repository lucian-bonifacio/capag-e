# 05 - Pressoes complementares de caixa

## Objetivo

Definir itens que reduzem o `ROA Final` por pressao financeira relevante.

## Itens minimos

- fornecedores vencidos;
- parcelamentos;
- contingencias;
- divida fiscal;
- divida trabalhista;
- mutuos;
- intercompany.

## Regras

1. Podem vir de contas patrimoniais.
2. Devem ser segregados do resultado operacional puro.
3. Devem ter rastreabilidade.
4. Quando materiais, devem exigir evidencia.
5. Devem reduzir `ROA Final`.

## Integracao com Modulo 04

Pressao material deve criar evidencia com:

- `method_component = ROA`;
- tipo de ajuste coerente;
- justificativa do analista;
- status documental.

## Criterio de sucesso

Pressoes complementares nao devem ser misturadas silenciosamente com despesa operacional.
