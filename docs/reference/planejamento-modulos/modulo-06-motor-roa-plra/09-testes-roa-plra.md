# 09 - Testes ROA + PLRA

## Testes obrigatorios

1. Calcular ROL com receita, deducoes e tributos.
2. Calcular ROA com custos e despesas.
3. Aplicar resultado financeiro com sinal correto.
4. Aplicar pressoes complementares.
5. Bloquear ROA por conta sem regra.
6. Gerar pendencia para despesa condicional material.
7. Combinar `ROA + PLRA`.
8. Impedir `CAPAG-E` sem `PLRA`.

## Areas afetadas

- `io`
- `domain`
- `engine`
- `assets`
- `api`
- `export`
- `frontend`

## Criterio de sucesso

O caminho `ROA + PLRA` deve ser independente, auditavel e compativel com o contrato da `CAPAG-E`.
