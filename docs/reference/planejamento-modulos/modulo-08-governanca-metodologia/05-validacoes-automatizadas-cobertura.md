# 05 - Validacoes automatizadas e cobertura

## Objetivo

Planejar validacoes para assets e metodologia.

## Validacoes recomendadas

1. Verificar colunas obrigatorias dos assets.
2. Verificar tratamentos permitidos.
3. Verificar componentes referenciados existentes.
4. Verificar regras sem componente quando deveriam ter.
5. Listar codigos referenciais com sobreposicao de prefixos.
6. Gerar relatorio de cobertura metodologica.

## Relatorio de cobertura

Deve listar:

- quantidade de regras PLRA;
- quantidade de regras DFC;
- quantidade de regras ROA;
- regras condicionais;
- regras sem politica de desagio;
- componentes sem regra;
- regras sem teste associado.

## Criterio de sucesso

Erro metodologico estrutural deve ser detectado antes de afetar resultado ou laudo.
