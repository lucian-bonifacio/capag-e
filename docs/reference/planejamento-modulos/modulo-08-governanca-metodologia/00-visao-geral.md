# 00 - Visao geral do Modulo 08

## Objetivo

Planejar governanca de manuais, specs, assets metodologicos, testes e comportamento real do sistema.

## Fonte principal

- `docs/specs/spec-12-governanca-manuais-metodologia.md`

## Papel do modulo

Este modulo reduz divergencia entre:

- documentacao;
- metodologia;
- assets;
- codigo;
- testes;
- laudo.

## Relacao com Modulo 00

O Modulo 00 define arquitetura e governanca base.

O Modulo 08 executa a governanca metodologica final, depois que os modulos funcionais estiverem planejados.

## Por que fica no fim

Porque `AGENTS.md`, `CLAUDE.md`, `README.md`, skills, specs e manuais precisam refletir a arquitetura final, nao uma etapa intermediaria.

## Escopo

- matriz de rastreabilidade;
- status de specs/manuais/assets;
- validacoes automatizadas;
- relatorio de cobertura metodologica;
- changelog metodologico;
- versao da metodologia usada no laudo.
