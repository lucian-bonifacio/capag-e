# 07 - UI de governanca metodologica

## Objetivo

Planejar se a governanca metodologica tera alguma visibilidade na interface.

## Escopo opcional

A UI pode mostrar:

- versao metodologica atual;
- alertas de asset incompleto;
- status parcial dos motores;
- data da metodologia;
- caminho dos assets usados.

## Fora do escopo

- Administracao livre de metodologia pelo usuario final.
- Edicao de regras de calculo pela interface.
- Publicacao formal de manual.

## Rotas possiveis

- `/governanca/metodologia`
- `/governanca/cobertura`

## Criterio de sucesso

O usuario pode conferir a versao metodologica usada sem poder alterar regra critica pela UI.
