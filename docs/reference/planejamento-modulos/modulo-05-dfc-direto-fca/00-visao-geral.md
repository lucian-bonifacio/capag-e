# 00 - Visao geral do Modulo 05

## Objetivo

Evoluir o `FCO` atual para uma `DFC` direta completa e apurar `FCA`.

## Fonte principal

- `docs/specs/spec-09-dfc-direto-completo-fca.md`

## Papel do modulo

Este modulo implementa o caminho:

```text
CAPAG-E = PLRA + FCA
```

Ele nao deve confundir o `FCO` atual com `FCA` completo.

## Relacao com modulos anteriores

- Modulo 01 calcula o `FCO` atual a partir de lancamentos.
- Modulo 03 define que `FCO` e apenas componente operacional do `FCA`.
- Modulo 04 define evidencias para fluxos materiais.

## Entregas principais

- Tabela metodologica DFC.
- Componentes DFC direto.
- Classificacao de fluxos operacionais, investimento e financiamento.
- Auditoria por lancamento.
- Pendencias de classificacao.
- Valor de `FCA`.
- Status final ou parcial.

## Fora do escopo

- Motor `ROA`.
- Geracao textual de laudo.
- Upload de DFC externa.
