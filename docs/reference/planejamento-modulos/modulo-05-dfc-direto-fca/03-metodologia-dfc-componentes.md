# 03 - Metodologia DFC e componentes

## Objetivo

Planejar assets internos para classificar DFC pelo metodo direto.

## Assets futuros

- `src/capag/assets/methodology/tabela_metodologica_dfc.csv`
- `src/capag/assets/methodology/componentes_dfc_direto.csv`

## Atividades DFC

- `operacional`
- `investimento`
- `financiamento`
- `nao_classificado`

## Componentes minimos

### Operacional

- recebimentos de clientes;
- pagamentos a fornecedores;
- pagamentos a empregados;
- tributos operacionais;
- outros fluxos operacionais recorrentes.

### Investimento

- compra de imobilizado;
- venda de imobilizado;
- aquisicao de participacoes;
- venda de participacoes;
- aplicacoes/resgates nao equivalentes a caixa.

### Financiamento

- captacao de emprestimos;
- amortizacao de principal;
- juros classificados como financiamento;
- aumento/reducao de capital;
- distribuicao de lucros/dividendos.

## Criterio de sucesso

A metodologia deve classificar por codigo referencial, com status claro quando nao houver regra.
