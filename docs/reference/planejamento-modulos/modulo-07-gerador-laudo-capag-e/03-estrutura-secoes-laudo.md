# 03 - Estrutura e secoes do laudo

## Objetivo

Definir secoes obrigatorias do laudo.

## Secoes

1. Identificacao.
2. Contexto do pedido.
3. Metodo de calculo.
4. Base de dados.
5. `PLRA`.
6. `FCA`, quando usado.
7. `ROA`, quando usado.
8. Evidencias e justificativas.
9. Resultado `CAPAG-E`.
10. Ressalvas e limitacoes.

## Dados minimos

- empresa;
- CNPJ;
- exercicios;
- data de emissao;
- responsavel;
- ECDs usadas;
- metodo;
- formula;
- valores;
- bloqueios;
- ressalvas;
- evidencias materiais.

## Criterio de sucesso

O laudo deve falar a linguagem da PGFN sem perder rastreabilidade tecnica.
