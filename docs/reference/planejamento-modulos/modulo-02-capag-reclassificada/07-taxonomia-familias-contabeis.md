# 07 - Taxonomia de familias contabeis

## Objetivo

Definir a taxonomia intermediaria usada antes do mapeamento para codigo referencial.

## Ponto de decisao

Decidir o conjunto MVP de familias e a lista de expansao futura.

## Familias MVP

1. Bancos e disponibilidades.
2. Clientes.
3. Fornecedores.
4. Adiantamento a fornecedores.
5. Imobilizado.
6. Despesas operacionais.
7. Receitas.

- Custo;
- Despesa;
