# 06 - Design system do frontend

## Objetivo

Criar uma especificacao visual separada da arquitetura tecnica.

## Ponto de decisao

Decidir se sera criada `docs/specs/spec-16-frontend-design-system.md`.

## Arquivos correlatos

- `web/src/styles/index.css`
- `web/src/App.jsx`
- `web/src/components/`
- `tests/design-previews/`

## Detalhamento

- Documentar identidade visual atual.
- Definir padroes para:
  - botoes;
  - icon buttons;
  - grids;
  - filtros;
  - cards;
  - barras de resultado;
  - drill-downs;
  - inputs monetarios e percentuais;
  - estados vazios;
  - alertas.
- Definir regras de responsividade.
- Definir quando criar previews em `tests/design-previews/`.

## Entregavel

Spec de design do frontend.

## Criterios de sucesso

- Design e arquitetura deixam de ficar misturados.
- Futuras telas seguem padrao consistente.
